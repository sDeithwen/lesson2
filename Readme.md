## TestApp - Тестовое Python-приложение для Docker
Простое консольное приложение на Python для демонстрации сборки Docker-образа, контекста сборки и работы контейнера.
### Описание
Приложение бесконечно выводит сообщение "Контейнер запущен и работает" каждые 5 секунд с логами в формате Loguru. Идеально для:
- Тестирования docker build и docker run.
- Проверки .dockerignore (логи в stdout, без файлов на диске).
- Демо в зуминаре/курсе по Docker основам.
### Требования
- Python 3.8+
- loguru (установка: pip install loguru)
### Локальный запуск
```
bash
pip install loguru
python app.py

```
Вывод:
```
2026-03-09 12:00:00 | INFO | Приложение стартовало
2026-03-09 12:00:05 | INFO | Контейнер запущен и работает
...
```

### requirements.txt
```
loguru==0.7.2
```
